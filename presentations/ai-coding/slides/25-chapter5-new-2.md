# 新项目启动（场景3-4）

<v-clicks>

## 场景3：接口契约先行

先用 AI 生成完整 API 契约，双方基于契约独立开发

**审查要点：**
- 字段命名是否一致
- 错误码是否规范
- 分页过滤等通用模式是否符合团队标准

## 场景4：原型快速验证

快速生成可交互原型，用于和 PM 或业务方确认方向

<Admonition type="warning" title="重要">

**原型代码是一次性的**

确认方向后，基于正式技术设计重新实现

不要在原型上"长"出正式产品

</Admonition>

</v-clicks>
