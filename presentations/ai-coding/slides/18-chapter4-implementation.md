# 实现阶段

<v-clicks>

## AI 主要做四件事

1. **代码生成** - 根据设计文档生成代码
2. **注释生成** - 给代码加注释、写文档
3. **测试生成与执行** - 生成测试用例并执行
4. **多层次测试** - 单元、集成、系统测试

</v-clicks>

<v-clicks>

## 两个重要 Skills

### vue-best-practices

Vue 开发规范：状态可预测、数据流显式、组件小而专注

### test-driven-development

测试驱动开发：**没有失败的测试，就不写生产代码**

先写测试 → 看它失败 → 写最小代码让它通过 → 重构

</v-clicks>
