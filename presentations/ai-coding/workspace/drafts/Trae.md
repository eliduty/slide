# TRAE：字节跳动 AI 原生 IDE

## 一、产品简介

TRAE 是字节跳动于 2025 年推出的 AI 原生集成开发环境（AI IDE），定位为"智能协作 AI IDE"，旨在通过深度融合的大语言模型能力，将开发者从"工具使用者"转变为"AI 协同创新者"。

**核心口号**：从 Copilot（副驾驶）升级为 Autopilot（自动驾驶）

**发展历程**：
- 2025年1月19日：国际版发布，内置 GPT-4o、Claude-3.5-Sonnet
- 2025年3月3日：国内版发布，中国首个 AI IDE，搭载 Doubao-1.5-pro，支持切换 DeepSeek-R1&V3
- 2025年11月：国际版移除 Claude 模型（因 Anthropic 服务条款调整），转向 GPT-5、Gemini-2.5-Pro 等模型

## 三、核心交互模式

### 1. IDE 模式 = Chat + Builder + 代码编辑

**Builder 模式**：
- 从0到1构建完整项目
- 用户通过自然语言描述需求，AI 自动拆解任务并生成代码框架
- 支持跨文件修改、运行终端命令
- 示例：输入"生成一个贪吃蛇应用"，自动创建所有文件并完成代码

**Chat 模式**：
- 侧边对话，解答代码疑问、讲解代码仓库、生成代码片段、修复错误
- 内嵌对话（Inline Chat）：编码过程中随时唤起，保持编码心流

### 2. Solo 模式 = 独立 AI 工程师

**定位**：自主执行复杂开发任务的 AI 工程师

**能力闭环**：理解需求 → 自主规划 → 调用工具（终端、Git、数据库）→ 独立完成编码、调试、测试全流程

**关键指令**：

| 指令 | 功能 | 适用场景 |
|------|------|---------|
| `/spec` | 让 AI 把模糊需求变成清晰、完整、可开发的详细规格书 | 需求模糊、怕 AI 理解错 |
| `/plan` | 根据需求生成一步步的开发任务清单，按顺序执行 | 需求已清楚，想让 AI 自动按步骤开发 |

## 四、核心能力

### 1. 智能代码生成与优化
- 通过自然语言描述生成代码片段
- 代码补全、重构建议、优化代码质量
- 支持 Python、JavaScript、Java、Go、C++、Rust 等 20+ 编程语言

### 2. 多模态交互
- 支持上传图片（如设计稿、草图）生成代码
- AI 精准解析后生成对应前端代码

### 3. 全局上下文分析
- 深度理解代码库和开发环境
- 提供精准的 Bug 修复和代码优化建议

### 4. 模型自定义
- 支持切换多种内置大模型
- **支持通过 API Key 接入自定义模型**，包括：
  - 国际服务商：OpenAI、Anthropic、Gemini、xAI、OpenRouter 等
  - 国内服务商：DeepSeek、火山引擎、阿里云、腾讯云、智谱 GLM、Kimi、MiniMax 等
  - 本地模型：Ollama Cloud

### 5. VS Code 完美兼容
- 一键导入 VS Code 的配置、快捷键和海量插件
- 无缝迁移，零学习成本

## 六、使用流程

```
需求描述 → /spec 定规格 → /plan 定计划 → 确认 → AI 自动执行 → 验收/修正 → 迭代
```

**发指令技巧**：场景 + 功能 + 技术 + 约束 + 细节

**不好的指令**：做个登录页

**好的指令**：
> 做一个移动端适配的用户登录页面，用 Vue3 + Vant，包含手机号 + 验证码登录、记住我、忘记密码、第三方微信登录；表单实时校验，错误提示友好；配色主色 #1677ff；要求 100% 还原常见电商 App 登录页交互；代码要规范、可维护、带注释。

## 七、优势与不足

### 优势
- **完全免费**：国内版所有功能免费，性价比极高
- **丰富的模型选择**：内置多款国产顶尖模型，支持 API Key 接入自定义模型
- **原生中文**：界面、代码注释均为中文，降低学习门槛
- **低门槛**：适合初学者快速上手
- **高效率**：显著缩短项目筹备周期
- **MCP 协议支持**：内置 MCP 市场，可灵活调用外部工具和服务

### 不足
- **复杂项目支持有限**：复杂项目仍需开发者手动调整
- **平台兼容性**：目前仅支持 macOS 和 Windows，Linux 版本尚未发布
- **国际版模型变化**：2025年11月起国际版已移除 Claude 模型

## 九、总结

TRAE 作为国内首个 AI 原生 IDE，在提升开发效率、降低技术门槛方面展现了显著优势。其核心价值在于：

1. **从 Copilot 到 Autopilot**：不仅是代码补全，而是能够自主完成开发任务
2. **结构化工作流**：`/spec` → `/plan` → 执行，让 AI 编码可预测
3. **本土化优势**：完全免费、原生中文、国内直连
4. **灵活的模型生态**：内置多款国产模型，支持 API Key 接入自定义模型
5. **MCP 协议集成**：内置 MCP 市场，可扩展工具能力

对于国内开发者而言，TRAE 是体验 AI 原生 IDE 的最佳入门选择。
