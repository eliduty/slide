# Skill 的意义

<div class="p-4 bg-blue-50 dark:bg-blue-900/20 rounded-lg border-l-4 border-blue-500 mt-4">

**从"对话"到"结构化工作流"**

- 传统 AI：纯对话式交互，结果不可预测
- Skill：结构化任务执行，结果可预期

</div>

### 可复用、可组合

<div class="text-sm mt-2 space-y-1">

- 一个 Skill 可以在多个场景中使用
- 多个 Skill 可以组合完成复杂任务
- 形成"能力积木"体系

</div>

</div>

<div class="text-sm mt-2 space-y-1">

- 从"人写代码"到"人定义规范，AI 执行"
- 开发者角色转变：从执行者到设计者
- 重新审视开发流程的契机

</div>
